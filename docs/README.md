## 使用建议

- 看项目是否值得做：优先看 `README.md`、`business-product-optimization-v3.md`
- 看商业和价值闭环：优先看 `README.md`、`roi-model-template.md`
- 看软硬件与 AI 能力：优先看 `hardware-ai-solution.md`
- 看项目壁垒和单店经济：优先看 `business-moat-and-unit-economics.md`
- 看技术落地方式：优先看 `edge-ai-architecture.md`
- 看商业化节奏和产品包：优先看 `business-roadmap-and-packaging.md`
- 看实施和交付：优先看 `implementation-blueprint.md`、`paid-pilot-playbook-8w.md`
- 做产品规划：优先看 `PRD-V1.md`
- 分直营/加盟方案：优先看 `commercial-plan-direct-vs-franchise.md`
