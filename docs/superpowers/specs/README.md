## Fast Start

If the team is starting now, use this quick sequence:

1. Lock product scope with Workbench + Action Card specs.
2. Confirm role flows with Homepage Flow spec.
3. Align UX wording with Interaction Copy spec.
4. Track engineering progress with MVP Delivery checklist.
5. Run UAT and baseline setup before pilot launch.
6. Use weekly deck + risk register during pilot.
7. Use go-live gate checklist for launch decision.
